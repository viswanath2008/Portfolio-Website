About

MV

I turn complex problems into efficient execution — no fluff, just results. That's the principle behind everything I build, from backend automation scripts to interactive front-end interfaces.

My work lives on GitHub, where I build and ship in public. Recent projects include SkyPulse, a vanilla JavaScript weather app with dynamic sky animations that earned community recognition, and a fully offline Python Library Management System built with Tkinter — demonstrating end-to-end ownership from database logic to graphical interface.

I'm currently expanding from Python and automation into full-stack web development — JavaScript, React, and modern tooling — while documenting the entire process. You can follow that journey and connect with my professional network on LinkedIn, where I share progress, lessons, and the occasional detour.

Open to engineering challenges where performance is the only metric that matters.
